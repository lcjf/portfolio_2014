    • Making life easier with Compass and SCSS

    A brief example of how you can utilise this CSS Authoring Framework to your advantage

    Gradients

    A linear gradient is just one example, this will also generate all the necessary vendor prefixes.

    Example
    @include background-image(linear-gradient($mine-shaft, $mine-shaft - 100));

    Border Radius

    Quite an obvious one; using this include will automatically generate all the necessary vendor prefixes when using border-radius.

    Example
    @include border-radius(5px);

    Transitions

    This is a huge time saver considering the amount of vendor prefixes that are required. The example below fades in all properties, using the 'ease-in' transition, over the space of 200ms.

    Example
    @include transition-property(all);
@include transition-duration(.2s);
@include transition-timing-function(ease-in);
&:hover {
    color: $mine-shaft - 100;
    background: darken($golden-rod, 15%);
}

    Sprites

    Another brilliant feature of Compass. This will automatically generate both your image sprite (based on a folder of images) and your CSS.

    Create a relevantly named folder inside your images folder, and copy all your images in to it. then add the below line of code to your main stylesheet where you have declared all your imports (typically style.scss stylesheet.scss or application.scss).

    @import "your_folder_name/*.png";

    You can also use @include to define select images from your sprite.

    .name_of_image {
    @include your_folder_name(name_of_image);
}

    Advanced Sprites

    Icon Sprites can be further customised.

    $icons-spacing: 20px;
$icons-flickr-spacing: 80px;
$icons-layout: horizontal;
$icons-layout: diagonal;
$icons-layout: smart;

    Hover states

    If you want to create a hover state for one of your sprites, all you have to do is create another image with the same name, but append '_hover' to the end e.g. 'facebook_hover.png'. Again, the styles are automatically generated.

    Horizontal and vertical lists

    This will automatically generate the styles required for your lists and their child elements.

    .lists ul {
    @include horizontal-list;
    a {
        display: block;
        width: 48px;
        height: 48px;
    }
}

    Vertical Rythm

    On the Web, vertical rhythm – the spacing and arrangement of text as the reader descends the page – is contributed to by three factors: font size, line height and margin or padding. All of these factors must be calculated with care in order that the rhythm is maintained.

    Source: http://24ways.org/2006/compose-to-a-vertical-rhythm

    The mixin and variables must be set for you to establish your vertical rhythms

    @include establish-baseline;
$base-font-size: 16px;
$base-line-height: 24px;

    These mixins further demonstrate how you can customise your vertical rythm:

    .paragraph-one {
    @include leader();
}
.paragraph-two {
    $default-rhythm-border-style: dashed;
    @include rhythm-borders(2px);
}
.paragraph-three {
    @include adjust-font-size-to(20px);
}
.paragraph-four {
    @include trailer();
}
